git log operation
---------------------------------------
####

    $ git log --stat
    $ git log --shortstat


    $ git log --oneline --graph --decorate --all


    $ git log --author='Iverson'


    $ git log --after='2015-02-18 9:30' --before='2015-02-25 22:30'


    # show specific log of file or folder.
    $ git log -- .
